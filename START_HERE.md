## ‚ùì Having Problems?

### "Python is not installed" error
- Download Python from: https://www.python.org/downloads/
- During installation, **CHECK the box** that says "Add Python to PATH"
- After installing Python, run `INSTALL.bat` again

### Can't find INSTALL.bat
- Make sure you extracted all files from the ZIP folder
- Look in the folder named "GAYBECK STARKIDS SMS"

### Application won't start
- Try restarting your computer
- Make sure you completed the installation
- Check your Desktop for the "Gaybeck Starkids Academy" icon
