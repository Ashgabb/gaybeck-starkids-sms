## If Icon Not Showing

**Try these in order:**

1. **Refresh Desktop**
   - Right-click desktop → Refresh
   - Or press F5

2. **Restart Explorer**
   - Already done, but can repeat:
   ```powershell
   Stop-Process -Name explorer -Force
   Start-Process explorer
   ```

3. **Manual Fix**
   - Right-click shortcut → Properties
   - Click "Change Icon"
   - Browse to: `sms_icon.ico`
   - Select and click OK twice

4. **Verify Files**
   ```bash
   python verify_icon_setup.py
   ```
