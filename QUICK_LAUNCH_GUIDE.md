## How to Launch the App

### Method 1: Desktop Icon (Recommended)
**Easiest way to launch:**

1. Look on your **Desktop**
2. Find the icon labeled **"Gaybeck Starkids SMS"**
3. **Double-click** the icon
4. Wait 5-10 seconds for the login window to appear
5. Enter login credentials (see below)

### Method 2: Windows Start Menu
**Alternative method:**

1. Press the **Windows key** on your keyboard
2. Type: **Gaybeck**
3. Press **Enter** or click the application entry
4. Login window will appear in 5-10 seconds

### Method 3: Command Line
**For technical users:**

```powershell
cd "c:\Users\User\Desktop\GAYBECK STARKIDS SMS"
```

**Try these steps:**

1. **Wait longer** - Sometimes takes 15-20 seconds on first launch

3. **Try again** - Close everything and try launching again

## Troubleshooting

### "Python is not installed" Error
Solution: Download Python 3.13+ from python.org and add to PATH

### Login fails with "Invalid credentials"
Solution: Use exactly these credentials:
- Username: `admin` (lowercase)
- Password: `admin123` (exact case)

### App starts but nothing shows
Solution: 
1. Wait up to 20 seconds on first launch
2. Check if window is behind other windows
3. Alt+Tab to find the application window

### Database Error
Solution:
1. Verify database file exists: `school_management.db`
2. File should be in: `C:\Users\User\Desktop\GAYBECK STARKIDS SMS\`
3. Database should have 41 tables
